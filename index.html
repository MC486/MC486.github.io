Word Strategy AI Project

This project implements an AI system that can play a word strategy guessing game, combining multiple AI techniques including Monte Carlo Tree Search, Naive Bayes classification, and Q-Learning. The system learns from its gameplay to improve its strategy over time.

Key Features

AI Components

Monte Carlo Tree Search - Strategic decision-making for word selection
Naive Bayes Classifier - Pattern recognition and move prediction
Markov Chains - Word generation and sequence prediction
Q-Learning - Reinforcement learning for strategy optimization

def play_game(self, target_word: str) -> List[str]:
    """Play a game of Word Strategy with the given target word."""
    game_state = GameState()
    moves = []
    
    while not game_state.is_game_over():
        # Get AI's move
        move = self.get_move(game_state)
        moves.append(move)
        
        # Update game state
        feedback = game_state.make_guess(move)
        self.update_state(feedback)
        
    return moves
